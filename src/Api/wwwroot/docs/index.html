ApplyRush API — Overview

Base URL: https://api.applyrush.com/api/v1

Auth: Bearer JWT in Authorization header. Obtain via POST /authenticate/login or /authenticate/google.

Worker endpoints require a separate X-Api-Key worker header instead of JWT.

Admin endpoints require the Administrator role in the JWT claims.

Async flows: Some actions publish a RabbitMQ message that is consumed by the Worker service. These are noted per endpoint.

Pagination: Paginated endpoints accept pageNumber and pageSize in the body. Response shape is { items: [], totalCount, pageNumber, pageSize }.

Resume

Uploading a resume publishes a ResumeUploadedMessage to RabbitMQ. The Worker consumes this, downloads the file, calls Claude claude-sonnet-4-5 to extract structured data, and merges the results into the user's profile (skills, work experiences, education, contact info).
